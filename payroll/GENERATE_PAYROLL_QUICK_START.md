# Quick Start: Generate Payroll from Profile

## 🚀 3-Step Process

### Step 1: Access Profile
```
Payroll Menu → Payroll Profiles → [Select Profile] → "Generate Payroll" Button
```

### Step 2: Configure Run
- **Run Name**: Auto-filled (e.g., "January 2025 - Regular Payroll")
- **Pay Period**: Set start and end dates
- **Payment Date**: When employees will be paid
- **Notes**: Optional notes about this run

### Step 3: Select Personnel
Choose ONE method:
- ✅ **All Active** - Include everyone (default)
- 🏢 **By Department** - Select specific departments
- 💼 **By Designation** - Select specific positions  
- 📋 **By Status** - Select employment statuses
- 👤 **Custom** - Manually pick individuals

### Step 4: Generate
Click **"Generate Payroll Run"** → Wait for processing → View results

---

## 📊 What Gets Created?

For each personnel in the run:

**Income Items** (from profile)
- Basic Pay
- Allowances
- Bonuses
- Other earnings

**Deduction Items** (from profile)
- SSS/GSIS
- PhilHealth
- Pag-IBIG
- Tax
- Loans
- Other deductions

**Calculations**
```
Gross Pay = Sum of all income
Total Deductions = Sum of all employee deductions
Net Pay = Gross Pay - Total Deductions
```

## ⚡ Quick Tips

### ✅ DO:
- Review profile items before generating
- Double-check dates
- Use descriptive run names
- Test with small groups first

### ❌ DON'T:
- Generate duplicate runs for same period
- Use overlapping pay periods
- Generate without reviewing totals
- Skip validation messages

---

## 🔧 Troubleshooting

### "No personnel found"
→ Selection criteria too restrictive  
→ Try "All Active Personnel"

### "Profile not found or inactive"
→ Check profile status  
→ Reactivate profile if needed

### "Missing required fields"
→ Fill in run name and dates  
→ All required fields marked with *

### Generation takes too long
→ Normal for 100+ personnel  
→ Wait up to 5 minutes  
→ Don't refresh page
