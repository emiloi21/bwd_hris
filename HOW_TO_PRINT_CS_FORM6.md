# HOW TO PRINT CS FORM NO. 6 (APPLICATION FOR LEAVE)

## 📍 WHERE TO FIND THE PRINT BUTTON

### Option 1: From Leave Application Page (Recommended)

**Step-by-Step:**
1. **Navigate to Leave Application page**
   - From Leave Card page: Click **Quick Options** → **View & Print Leave Applications**
   - OR directly go to: `leave_application.php?personnel_id=XXX`

2. **Find the leave application to print**
   - Look for the table with all submitted leave applications
   - Each row has an ellipsis menu (⋮) button

3. **Click Print**
   - Click the **⋮** button on the leave application row
   - Select **"Print CS Form No. 6"**
   - Print preview modal will open

4. **Review and Print**
   - Review the form preview
   - Click **Signatories Settings** (first time only) to configure
   - Click **Print** button to print or save as PDF

---

### Option 2: After Submitting Leave Application

**Step-by-Step:**
1. **Submit a leave application** from Leave Card page
2. **Go to Leave Application page** (via Quick Options menu)
3. **Follow steps 2-4** from Option 1 above

## ❓ COMMON QUESTIONS

**Q: Why can't I print directly from Leave Card page?**
**A:** The Leave Card page shows your leave credits (earned/used). To print the actual **Application for Leave (CS Form No. 6)**, you need to go to the Leave Application page where submitted applications are listed.

**Q: I submitted a leave application but can't find it?**
**A:** 
1. Click **Quick Options** → **View & Print Leave Applications**
2. You'll be redirected to the Leave Application page
3. Your submitted applications will be listed in the table

**Q: Where is the ellipsis (⋮) button?**
**A:** It's in the first column (ACTION column) of each row in the Leave Application table. It looks like three vertical dots.

**Q: Do I need to configure signatories every time?**
**A:** No! Configure signatories **once** (first time), and they'll be saved for all future printouts.

**Q: Can I print without submitting the application?**
**A:** No. You must first submit the leave application, then access the print feature from the Leave Application page.

## 🆘 TROUBLESHOOTING

**Problem: Can't find "View & Print Leave Applications" option**
**Solution:** 
- Refresh the page (Ctrl+F5)
- Check you're on the Leave Card page
- Look in the **Quick Options** dropdown menu (top navigation)

**Problem: Print button not showing**
**Solution:**
- Make sure you're on the **Leave Application page** (not Leave Card page)
- Check the ACTION column (first column with ⋮ button)
- Ensure there are submitted leave applications in the table

**Problem: No leave applications showing**
**Solution:**
- Submit a leave application first from Leave Card page
- Use the "Leave Application" button on Leave Card page
- Fill out the form and click Submit
